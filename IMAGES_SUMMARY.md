## 🚀 Quick Answer: How to Add Images

### Method 1: Use the Files
1. Put your image file in the `images` folder
2. Open `index.html` in any text editor
3. Find line 181 (the Impact section)
4. Copy this code block:
```html
<div class="impact-item">
    <img src="images/YOUR_IMAGE_NAME.jpg" alt="Educational impact - Description">
    <p class="impact-caption">Your caption text here</p>
</div>
```
5. Paste it inside the `<div class="impact-gallery">` section (there's a template already there!)
6. Change `YOUR_IMAGE_NAME.jpg` to your actual filename
7. Change the caption text
8. Save and open index.html in your browser to see it!

### Method 2: Read the Guide
Open the new `ADD_IMAGES_GUIDE.md` file for detailed instructions with examples!

## 🎯 Example

Let's say you have a photo called `school-visit-2024.jpg`. Here's what you do:

1. **Copy the file** to the `images` folder
2. **Open** `index.html`
3. **Find** the Impact section (around line 181)
4. **Add** this code:

5. **Save** and reload your browser!
